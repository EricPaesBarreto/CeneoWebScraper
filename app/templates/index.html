  • Purpose of the application

    The purpose of this application is to scrape data from the popular online-retail store ceneo.pl and provide interesting statistics for products available on the site.

    This includes graphs, as well as some basic descriptive statistics. Another purpose of this project was to learn the basics of web development using the Flask framework amongst other tools, and familiarise myself with how web-applications work.

  • Description of the application

    The application works in a very simple way, firstly, product data must be extracted, during this process statistical data and graphs are generated. You may access pre-extracted products on the products page.

    Extraction of products may be done on the extraction page, for this to properly work, you must input a valid product ID from the ceneo website. You can get a Product id by checking the url of the product on ceneo.pl.

    This program was made with an object oriented approach to dealing with the representation of products and opinions, with a mix of typical procedural programming.
